# TakeMe

Веб-приложение для организации совместных поездок сотрудников в офис.

## Стек
- Backend: NestJS, Prisma, PostgreSQL
- Frontend: React, TypeScript, Redux Toolkit (RTK Query), React Router
- Карты: Leaflet, OpenStreetMap, OSRM
